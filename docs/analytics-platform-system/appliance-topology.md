# <a name="appliance-topology---analytics-platform-system"></a>어플라이언스 토폴로지-분석 플랫폼 시스템
전체 분석 플랫폼 시스템에 대 한 요약 정보를 표시 합니다.  
  
이 페이지에는 해당 이더넷 및 Infiniband IP 주소를 포함 하 여 현재 어플라이언스의 모든 호스트 및 패브릭 노드에 대 한 읽기 전용 버전이 나열 됩니다.
